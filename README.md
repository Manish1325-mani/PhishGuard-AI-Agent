# 🛡️ PhishGuard-AI Security Agent

PhishGuard-AI is an advanced, conversational AI mentor specialized in **Cybersecurity** and **Ethical Hacking**. Built using Google's **Gemini 3 Flash** and **Streamlit**, it provides real-time forensic analysis of threats and malicious code.

## ✨ Key Features
